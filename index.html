Responsive design

A responsive design adjusts to the screen size being used while visiting a website. To ensure that your website is responsive you need to use HTML and CSS. By using the @media rule in CSS you can specify how different elements should be presented differently. For this to work properly you also need to work with percentage measurements rather than pixels when deciding sizes, in both your grid and your images. Since percentages are relative and not absolute values like pixels the grid and images becomes fluid. Try adjusting the width of this browser window and you'll see how the text and image changes!
